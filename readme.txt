------------ CAN project ------------ 

1. Project requirements
- Project read CAN message from another device then display to LCD
- Need two device: a CAN device is master display data to LCD
                   another CAN device is CAN node that collect data about car speed
- CAN device master use stm32f746 and CAN node use beagle bone black
